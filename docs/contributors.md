# Contributors

LibreMesh is a community and an open project, so anyone friendly is welcome.    
We encourage anyone using the LibreMesh to become part of this community.    
The software is maintained by some of its founders together with a [wide community](https://github.com/libremesh/lime-packages/graphs/contributors).
